Vim tips

by pratz

August 15, 2014
in Editor

Search and replace:

  • Search and replace in entire project

      # search and fill buffer
  args **/*.py

  # replace and update
  argdo %s/original/target/ge | update

  • Remove blank lines

      :g/^$/d

Profiling

  • Startup time profiling

      vim --startuptime vim.log

  • Complete profiling

      :profile start profile.log
  :profile func *
  :profile file *
  :profile pause
  # Note - be patient
